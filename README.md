# Kozu Obsidians Synthetic Data 

## Project Outlines
- Kozushima Obsidians (Kozu Obsidian) are spread across mainlan Japan in Periolithic to Yayoi period. This project conducts the synthetic simulation of the Kozu Obsidian share in the Jomon Honshu region.
- The Bayesian causal analysis is based on the [Statistical Rehinking 2026](https://github.com/rmcelreath/stat_rethinking_2026) by Richard McElreath.
- See [Overview.pdf](https://github.com/Mufurun/KOsyndata/blob/main/overview.pdf) for the details of priors and statistical model
